Medical Analyzer UI

医療用分析装置を想定し、操作性、視認性、 誤操作防止を考えながら制作しているWPF学習プロジェクトです。

プロジェクトの目的

React / TypeScriptで得たUI開発の知識を、 WPF / C#へ展開する過程を記録します。

現在の実装

  • 装置状態の表示
  • 検体操作パネル
  • 検体一覧の空状態
  • 操作可否に応じたボタン制御
  • ウィンドウサイズに追従するレイアウト

使用技術

  • C#
  • .NET 8
  • WPF
  • XAML
  • Visual Studio 2022

今後の予定

  • C#データモデル
  • データバインディング
  • MVVM
  • 入力検証
  • 測定状態と非同期処理
学習およびポートフォリオ用のデモプロジェクトです。
